# Hướng dẫn sử dụng ZaloCRM — Community Edition

Bộ ảnh hướng dẫn giao diện **ZaloCRM Community** (bản mã nguồn mở, AGPL-3.0), chụp trên hệ thống thật với **dữ liệu demo** (tổ chức *Clarity*).

> Mọi số liệu trên ảnh là **dữ liệu demo** phục vụ minh hoạ.
>
> Bản Community **không** có các tính năng Extension (tự động hoá Marketing đầy đủ, Lead Pool, Bot Auto…). Menu **Marketing** ở bản này chỉ gồm **Quét nhóm** và **Tệp khách hàng**.

## Đăng nhập demo

| Vai trò | Đăng nhập | Mật khẩu |
|---|---|---|
| Quản trị (admin) | SĐT `0901000001` hoặc email `nv1.demo@locnguyendata.com` | `Demo@1234` |
| Nhân viên/Sale | SĐT `0901000003`…`0901000030` | `Demo@1234` |
